# `data/`

Drop your input corpora here — one subfolder per corpus, each containing PDFs
or `.txt` files (e.g. `data/myproject/paper1.pdf`, `data/myproject/paper2.txt`).

Process a corpus with `python geordie_miner.py run data/<name>`. The contents
of this folder are gitignored — your papers stay local, only this README is
tracked.
